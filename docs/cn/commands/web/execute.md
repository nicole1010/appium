# 执行脚本

在页面中注入一段JavaScript，以便在当前选定框架的上下文中执行 (Web context). 运行 [mobile command](/docs/en/commands/mobile-command.md) (Native Context).

// Javascript
// webdriver.io example
var result = browser.execute(function(a, b, c, d) {
  // browser context - you may not access client or console
  return a + b + c + d;
}, 1, 2, 3, 4)

// node.js context - client and console are available
console.log(result); // outputs: 10

// wd example
await driver.execute('document.title');

## 描述

假定执行的脚本是同步的，并且评估脚本的执行结果将返回给客户端。

脚本参数定义了脚本以函数体形式执行， 该函数的返回值也将返回给客户端. 函数将调用提供的参数数组以及通过指定顺序获取的参数对象的值.

参数可以是任何JSON的基本类型, 数组, 或者 JSON 对象。 定义WebElement引用的JSON对象将转换为相应的DOM元素。 同样地, 脚本结果中的任何WebElements都将作为WebElement JSON对象返回给客户端。

（Native Context）中 `execute` , 详见 [Mobile Commands](/docs/en/commands/mobile-command.md).
